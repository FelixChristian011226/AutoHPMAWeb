# 自动甜蜜冒险

## 功能介绍 <Badge type="tip" text="Introduction" />

**自动甜蜜冒险**为`霜结燃烧火刑架`赛季限时活动`甜蜜冒险`的自动化任务。

- **使用场景**：甜蜜冒险匹配界面，或任务中任何阶段。
- **任务流程**：
  - 匹配阶段：自动点击匹配按钮。
  - 冒险过程：实时检测当前回合和步数，每回合在前12步点击前进，第13步点击返回。返回后默认点击预测怪物。
  - 结算过程：关闭结算页面，返回匹配页面。

- **终止条件**：手动终止。
